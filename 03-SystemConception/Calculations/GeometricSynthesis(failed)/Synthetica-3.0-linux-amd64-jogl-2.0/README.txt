Synthetica 3.0 for Win32, MacOSX, Linux, Solaris

Introduction
=========================
Synthetica is a Java program that is used to synthesize, analyze and simulate spatial linkages. Its' open architecture allows advanced users to add Java code to the system. Currently it runs on Windows, MacOSX and Linux and Solaris. The 3D graphics is based on JOGL, a java binding of OpenGL library provided by Sun Microsystem and SGI.

Instructions for End Users
=========================
To launch Synthetica, please either

double-click Synthetica.jar file

Or use the folloing command

java -jar Synthetica.jar
